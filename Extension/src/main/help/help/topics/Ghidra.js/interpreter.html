JavaScript Interpreter

JavaScript adds interactive JavaScript shells to Ghidra via Rhino, and also allows scripts to be written in JavaScript with full access to all of Ghidra's functionality.

This is accomplished using Rhino to bridge the Java APIs of Ghidra with a JavaScript environment.

Additional Help

For more information on the Rhino environment, such as how to interact with Java objects through a JavaScript interface, please refer to JavaScript wiki at https://p-bakker.github.io/rhino/
